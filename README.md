# ToolHub - 个人工具箱

一个使用 Vue3 + Element Plus 构建的现代化工具类网站。

## 技术栈

- Vue 3.4+
- Vue Router 4.2+
- Vuex 4.1+
- Element Plus 2.5+
- Vite 5.0+

## 项目结构

```
src/
├── assets/styles/     # 全局样式
├── components/        # 公共组件
│   └── layout/        # 布局组件
├── router/           # 路由配置
├── store/            # Vuex 状态管理
├── views/            # 页面视图
├── App.vue           # 根组件
└── main.js           # 入口文件
```

## 安装与运行

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产
npm run build

# 预览构建结果
npm run preview
```

## 功能特点

- 顶部导航栏
- 路由面包屑
- 响应式布局
- 现代化 UI 设计
